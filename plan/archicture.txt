# 🧠 Mental Model

## User Flow

```
User Action
   ↓
Task Updated
   ↓
Event Created
   ↓
Gamification Reacts
   ↓
Points / XP / Rewards Updated
```

---

# 🧩 Responsibility Separation

## Mobile App

Handles:

* UI
* Interaction
* Navigation
* Local state

---

## Supabase

Handles:

* Authentication
* Data persistence
* Permissions
* File uploads
* Realtime sync

---

## Event Layer

Acts as bridge between:

* Tasks
* Gamification
* Rewards
